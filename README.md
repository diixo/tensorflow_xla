tensorflow_xla
===================

# Build options

```
mkdir tensorflow_xla-build
cd tensorflow_xla-build
cmake -DCMAKE_BUILD_TYPE=Debug ../tensorflow_xla
cmake --build .
```

# Build

Build dependencies
- 

```
cmake -H. -Bbuild
```
